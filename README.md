# Conversation

API for managing chatbot interactions. Built with Node, Express, Firebase, and Dialogflow.

## Environment

You'll need the following environment variables defined:

- GCP_KEY_FILE
- GCP_CRED
- APP_NAME
- APP_ENV
- PROD_PROJECT_ID
- TEST_PROJECT_ID
- PORT
- AGENT_AVATAR_URL
